### Käyttöohje
- **1. Kloonaa github**
- **2. Mane hakemistoon, johon kloonasit sovelluksen**
- **3. Aja sovellus hakemistosta komennolla (python main.py)**
- **4. Komennot: new (lisää uusi artikkeli), list (näyttää tallennetut artikkelit), quit (sovellus sulkeutuu)**
